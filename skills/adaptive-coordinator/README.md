# adaptive-coordinator

Planned main Kybernetes skill.

This folder intentionally does not contain a `SKILL.md` yet. Kybernetes will create this skill after pressure scenarios prove the exact behavior it must teach.

Expected scope:

- adaptive task intake
- execution profile creation
- durable control record selection
- runtime binding selection
- human-in-the-loop steering
- same-chat, subagent, parallel-chat, or isolated-workspace routing
- impediment, escalation, decision, and learning capture
