# 038 - Short Consequential Decision Needs Architecture

## Failure

A short but irreversible decision proceeds from one perspective because the
governor incorrectly uses task length as the architect trigger.

## Expected Coordinator Behavior

- Trigger `kybernetes:loop-architect` because consequence and unresolved
  uncertainty are high.
- Generate independent lenses from the decision's actual uncertainty.
- Return a bounded execution contract and preserve disagreement.
